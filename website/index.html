Why this exists

A fork for people who want bookmark recall to feel serious.

FAQ

Quick answers for release confidence.

How do I install it?

Use the npm command shown above, then run ftx doctor and ftx sync.

Do I need a backend or account?

No. It stays local-first and stores your archive on your own machine.

Is this still a CLI?

Yes. The landing page is for the CLI, the repo, and the install path—not a hosted app.
